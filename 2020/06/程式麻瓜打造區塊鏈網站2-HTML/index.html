程式麻瓜打造區塊鏈網站2-HTML HTML

是一種網頁程式，他的架構是由標籤組成，這個標籤包著和，這兩個標籤又包著其他標籤，裡面主要放瀏覽器上標籤頁的標題title和其引入其他程式的連結，就是放要顯示在網頁的內容如文字、照片等，而標籤的結尾是，以下直接由程式架構重頭到尾說明: 1. 這行就只是說明這是HTML檔案 沒什麼功能 2. 這個箭頭加驚嘆號裡可以放註解 不會顯示 3.第一頁 title裡是分頁(tab)的標題 4.
個人檔案
h1裡包要顯示在網頁的文字 數字表示文字大小 最多到6 頭貼
p一樣包裹文字 顯示出來是一般文字 5.

hr/換行 6. img是圖片src是圖片位置 圖片直接放在你建立的含有index.html的資料夾內 src=”放檔名” 7.etherscan 查詢網站 a可以超連結文字 用herf=”網址”表示 target=”_blank”表示用新的分頁開啟 8.
male input 是輸入 type是種類 checkbox是多選盒 9. 是表單裡面放文字輸入框和提交按鈕 如下
text是一般的人字輸入框 只有一行
email只能輸入email形式
textarea是無限大的文字框 submit
button是按紐 class決定按紐的形狀等 onclick是只當他被點下會執行甚麼之後在JS會提到 以下是整個網頁程式的範例，將整個HTML碼複製起來貼到vscode裡你創建的index.html再用瀏覽器index.html打開就能看到效果了，再自己修改程式碼玩玩看，基本上就會使用了。 點選run pen後 按下左上角HTML按鈕可查看程式碼
