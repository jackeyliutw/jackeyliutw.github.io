程式麻瓜打造區塊鏈網站4-超強大chrome編輯網頁小工具VisBug

在排版網頁的時候每次都要一直調位置座標超級麻煩(應該是我學藝不精)，還好我發現一個chrome extention VisBug 能以滑鼠直接移動位置。

用法很簡單，首先將你要修改的網頁用chrome開啟，點擊visbug圖示，在左邊的欄位選擇第十個看起來像是方向鍵的圖示，再以拖移的方式調整你要修改的區域

再點visbug圖示，把它關掉，在空白處按右鍵，點檢查，到的標籤上按右鍵複製html程式碼，之後回到編輯器把整個程式碼覆蓋就完成了

當然visbug還有很多功能像是打文字、換顏色之類的，甚至可以直接別人的網頁來修改，建議去實做看看，它可以幫助你更了解整個網頁程式的各種功能。

程式麻瓜打造區塊鏈網站3-css/js和網頁特效

網頁程式語言可不是只有HTML一種，另外還有兩種常見的語言css和javascript，css是用來作美編的，能控制文字顏色和邊框等…JS則是控制網頁的動作等…

至於怎麼編寫呢?我這裡暫時就講不太多，我主要教各位如何將她加入網頁以及如何把別人寫好的程式放入自己的網頁裡，畢竟現在那麼多開放無版權的程式碼，隨便挑幾個拼湊起來也是能做出像樣的東西。

寫入css和js有以下幾種方式:

一.CSS:

1.Inline Style直接在標籤內添加CSS:

<p style=”color:yellow; font-size:2em”>CSS教學

2.Internal Style Sheet: 在標籤內統一管理樣式:

<head>
<style>
h1 {color:green;}
/* 把h1定義為綠色*/
style>
head>

3. Enternal Style Sheet — link: 引入CSS檔案，在head加入以下此行，href後面是放檔案名稱，如此，當打開index.html就會讀到css檔案，這個概念和img一樣

<head>
<link href=”./CSS檔名.css” rel=”stylesheet”>
head>

這方法要先在當初放有index.html的資料夾建立副檔名為.css的檔案，如下

上述各方法的衝突時的優先：Inline > Internal > Link/Import >Browser Default

二.JAVASCRIPT:

1.internal script:在標籤內插入script

<body>
<script>
JS程式碼
script>
body>

2.external:引入JS檔案，在body加入以下此行，href後面是放檔案名稱

<body>
<script src=”./檔名.js”>
script>
body>

像是上面這個codepen網站裡有很多特效，使用方法將html css js 分別複製下來使用，另外還有animista也很好用，如果有更厲害的資源歡迎留言分享

程式麻瓜打造區塊鏈網站-系列文1-網頁的編輯方法

廢話不多，我們直接實作。

首先請在隨便一個地方如桌面按右鍵>

新增資料夾>

在資料夾內再按右鍵新增文字文件>

重新命名，如改成index.html ，index是指網站的主頁，在當網址被點下的時候瀏覽器會讀取你資料夾，然後開啟index.html這個檔案

這裡需特別注意.html這東西叫做副檔名如果不會修改請搜尋”更改副檔名”

改完副檔名後直接點開會發現她會從瀏覽器開啟而不是記事本，這樣就可以算是一個網頁了，所以說網頁的概念很簡單，就是在電腦建立一個資料夾，放入網頁內容讓人來造訪，但是現在網頁沒上伺服器，只有你能造訪。

接下來就是編寫網頁了，要編寫網頁，需要用編輯器(IDE)來編輯，如上圖OPEN選項有顯示的編輯器有記事本、visual studio code，我推薦使用visual studio code來編寫。

安裝完後，回到你的網站資料夾，將檔案用vscode開啟(如上圖)就能開始編寫了。
