關閉迴響(從資料庫)

虛擬主機(cPanel)關閉迴響的步驟
就要先找到你的資料庫(Database)
->點擊phpMyAdmin ->找到wp_posts ->點選結構(structure)
再來將comment_status的預設值設為closed

關閉迴響(從外觀主題)

外觀 -> 主題編輯器 -> content.php ＆ page.php裡面
在用Ctrl+F尋找comments_template();並在前方使用//註解掉就關閉囉！
如下：

if ( comments_open() || get_comments_number() ) {
        //comments_template();
}
