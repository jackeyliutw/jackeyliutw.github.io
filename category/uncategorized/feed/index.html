]]> https://jackeyliutw.github.io/2020/06/30/14/01/feed/ 0 程式麻瓜打造區塊鏈網站2-HTML https://jackeyliutw.github.io/2020/06/30/14/00/ https://jackeyliutw.github.io/2020/06/30/14/00/#respond Tue, 30 Jun 2020 06:00:32 +0000 https://jackeyliutw.github.io/?p=17 Continue reading 程式麻瓜打造區塊鏈網站2-HTML]]> HTML是一種網頁程式，他的架構是由標籤組成，<html>這個標籤包著<head>和<body>，這兩個標籤又包著其他標籤，<head>裡面主要放瀏覽器上標籤頁的標題title和其引入其他程式的連結，<body>就是放要顯示在網頁的內容如文字、照片等，而標籤的結尾是</>，以下直接由範例說明:

  1. <!DOCTYPE html>

這行就只是說明這是HTML檔案 沒什麼功能 

2.<! -- -->

這個箭頭加驚嘆號裡可以放註解 不會顯示

3.<title>第一頁</title>
title裡是分頁(tab)的標題 

4.<h1>個人檔案</h1>
h1裡包要顯示在網頁的文字 數字表示文字大小 最多到6 

<p>頭貼<p/>
 p一樣包文字 顯示出來是一般文字 

5.<hr/>
hr/換行

6.<img src=”me.png” />
 img是圖片src是圖片位置 圖片直接放在你建立的含有index.html的資料夾內 src=”放檔名” 
 
 7.<a href=”https://rinkeby.etherscan.io/" target=”_blank”>etherscan 查詢網站</a>
 a可以超連結文字 用herf=”網址”表示 target=”_blank”表示用新的分頁開啟

8. <input type =”checkbox” name=”gender” value=”1"/>male input 是輸入 type是種類 checkbox是多選盒

9.<form>
是表單裡面放文字輸入框和提交按鈕 如下

<input type=”text” name=”name”>
 text是一般的人字輸入框 只有一行 

<input type=”email” name=”mail”>
email只能輸入email形式 

 <textarea name=”message”></textarea>
textarea是無限大的文字框 

<button class=”btn btn-default” onclick=”record()” type=”submit1">submit</button>
button是按紐 class決定按紐的形狀等 onclick是只當他被點下會執行甚麼之後在JS會提到
